1 Introduction

1.1 Purpose

The purpose of this document is to present a detailed description of the web application NUME_SITE based on the project M-PIC. The document will explain the purpose and features of the application, the interfaces, what the software will do and the constraints under which it must operate. This document is intended for the potential users of the software.

1.2 Document Conventions

This Document was created based on the IEEE template for System Requirement Specification Documents.

1.3 Intended Audience and Reading Suggestions

Users who wish to understand and learn how to use the application and its intended scope.

1.4 Product Scope

NUME_SITE was developed as an open source tool with which users can manage photos from different social media platforms in one place.
